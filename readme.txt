Format your web pages for printing. This plugin adds a page template that when selected, shows page breaks and print formatting. You can use a page builder like Elementor to visually style your page for printing then print to PDF to turn your website content into books and ebooks.

== Description ==
This plugin adds a page template that when selected shows how the page will print on 8.5 inch by 11 inch paper. The page breaks are shown and you can use a page builder like Elementor to visually style your page for printing then print to PDF to turn your website content into books and ebooks.

== Installation ==
1. Upload \"paper.zip\" to the \"/wp-content/plugins/\" directory.
2. Then activate this plugin.
3. Add the 8.5 Inch by 11 Inch Paper Template to a page.
